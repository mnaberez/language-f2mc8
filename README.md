# language-f2mc8

## Overview

This is an [Atom](https://atom.io) package that adds syntax highlighting for Fujitsu F2MC-8L and F2MC-8FX assembly language.  It supports the
[ASF2MC8](http://shop-pdp.net/ashtml/asf2mc.htm) assembler.
